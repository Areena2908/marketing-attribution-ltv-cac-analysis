# Marketing Attribution and LTV/CAC Analysis

## Project Overview
This project analyzes marketing channel performance using customer acquisition cost (CAC), lifetime value (LTV), LTV/CAC ratio, payback period, ROI, and monthly customer acquisition trends.

The goal is to identify which marketing channels generate the strongest business value and recommend where the next marketing dollar should be reallocated.

## Business Questions
- Which marketing channels acquire the most customers?
- Which channels have the lowest CAC?
- Which channels generate the highest LTV?
- Which channels have the strongest LTV/CAC ratio?
- Which channels recover acquisition cost fastest?
- Which channels should receive more or less marketing budget?

## Dashboard Features
- LTV/CAC Ratio by Channel
- CAC vs LTV Scatter Plot
- Payback Period by Channel
- Monthly Customers Acquired
- ROI by Channel

## Key Insights
- Organic Search, Referral, and Email showed the strongest LTV/CAC ratios.
- Referral and Organic Search generated high customer value with relatively low acquisition cost.
- Display had the weakest performance, with high CAC, low LTV, negative ROI, and the slowest payback period.
- Budget should be shifted toward Organic Search, Referral, and Email while reducing or closely monitoring Display spend.
